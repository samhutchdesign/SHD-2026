# SHD 2026 — Sam Hutch Design

A dynamic personal website for the freelance UX/product design & strategy practice, **Sam Hutch Design**.

## The idea

A short quiz (max 5 questions) walks each visitor through Sam's services and experience based on their answers, ending in:
1. a **personalized takeaway** they keep (a tailored "Design Recipe" / project-starter card), and
2. a **book-a-discovery-call** CTA.

Inspiration: [SOUR's cmd+zest](https://www.wearesour.studio/cmd-zest) — a playful-but-professional 5-question quiz with a personalized keepsake result.

## Positioning (short)

Lead with premium **product design & strategy** (Sam's strongest work — Doodles, NBA Top Shot, Amazon Alexa), while still routing "I just need a website" visitors to a simpler offer. Full stance in [`notes/positioning.md`](notes/positioning.md).

## Project phases

1. **Positioning & research** — strategy notes (this is done first; see `notes/`). ✅ drafted
2. **Quiz design & content** — 5-question flow, branching, result + takeaway copy.
3. **Design & mockups in Efecto** — full visual design before code.
4. **Build** — Next.js + TypeScript + Tailwind + Framer Motion; PostHog + Vercel Analytics; book-a-call integration.
5. **Polish & launch** — content, accessibility, perf, domain decision.
